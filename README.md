# hades_linkchecker
Tool for check a website

## How to run 🖥️️
1. Clone the repo
   ```sh
   git clone https://github.com/hadesyingcracker/Hades-linkchecker
   ```
2. Entry the path
   ```sh
   cd Hades-linkchecker
   ```
3. Run the app
   ```sh
   python hades_linkchecker.py
   ```
4. Before that you must put the link into link.txt
